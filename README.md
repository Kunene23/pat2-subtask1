# OVERVIEW
A Morse code is a telecommmunication method that encodes text characters as sequences of two signal durations : dots (short) and dashes (long)
# HISTORICAL CONTEXT
It was developed in the early 1830s and it is named after Samuel Morse,one of the several developers of the system
Morse's preliminary proposal for telegraph code was replaced by an alphabetic-based code developd by Alfred Vail,the engineer working with Morse.
The International Morse Code encodes the 26 basic Latin Letters A - Z ,one accented Latin Letter , the Indo-Arabic numerals 0 - 9 and some punctuation and messaging procedural signals (prosigns). The is no duration between upper and lower case letters. Each code symbol is formed by a sequence of dots and dashes.
# How It Works
- Dot (.) = short signal
- Dash (-) = long signal
- Letters separated by spaces, words by slashes or multiple slashes
# Examples
- Hello > ' .... . .-.. .-.. ---'
- World > ' .-- --- .-. .-.. -.. '
